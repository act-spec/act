# Documentation ACT

Cette page est la racine de la traduction française de la documentation
de démarrage. Elle montre comment Starlight et `@act-spec/plugin-astro`
gèrent ensemble le contenu multilingue : Starlight rend la version
française pour les humains, l'intégration ACT exporte le même contenu
dans le tree ACT pour les agents.

Reportez-vous à la [version anglaise](/) pour la liste complète des
pages disponibles.
