# Tinybox Docs

Tinybox is a small object-storage SDK. The docs are organized into a
short [Guide](./guide/getting-started.md) and a [Reference](./reference/configuration.md)
section.

This site is the canonical worked example for `@act-spec/plugin-vitepress`:
the same VitePress build that produces the human-facing HTML also writes a
machine-readable ACT (Agent Content Tree) under `.well-known/act.json` and
`/act/...`.

## What you will find here

- **Guide** — install Tinybox, send your first request, learn the model.
- **Reference** — every config field, every CLI flag, every endpoint.
- **Español** — a small Spanish locale demonstrates VitePress i18n flowing
  through to the ACT locale tree.
